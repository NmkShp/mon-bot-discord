# 1️⃣ Système de protection (Protect)
Le bot doit protéger le serveur contre les abus.

Fonctionnalités obligatoires :
- Anti raid
- Anti spam
- Anti bot
- Anti lien malveillant
- Anti mass mention
- Anti suppression massive de salons
- Anti suppression massive de rôles
- Anti ban/kick massif
- Logs complets de sécurité

Options avancées :
- Système de whitelist
- Système de blacklist
- Limitation d'actions administrateur
- Alertes dans un salon logs
- Auto-modération intelligente

Commandes administrateur pour configurer la protection.

---

# 2️⃣ Système de tickets avancé
Créer un panel de ticket interactif que l’administrateur peut envoyer dans un salon.

Fonctionnement :
1. L'utilisateur clique sur un bouton "Créer un ticket"
2. Le bot crée un salon privé
3. Dans le ticket, le bot envoie un message interactif avec un menu de sélection

Menu des services :

- Nitro Boost
- Server Boost
- Graphisme
- Comptes (ACC)
- Membres
- Tools
- Autre
- Questions / Support

Une fois sélectionné :
Le bot pose automatiquement des questions pour comprendre la demande.

Exemple :
- Quel service veux-tu exactement ?
- Quel est ton budget ?
- Délai souhaité ?
- Détails supplémentaires ?

Fonctionnalités supplémentaires :
- Bouton Fermer le ticket
- Bouton Claim ticket
- Bouton Ajouter un membre
- Bouton Transcription du ticket
- Logs des tickets
- Catégorie automatique

Interface :
- Utiliser embeds propres
- Boutons Discord
- Menus déroulants
- Design moderne
